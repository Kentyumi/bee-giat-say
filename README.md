# Bee Giặt Sấy

Web app login với bầy ong sống động, UI hiện đại. Dùng Vite + React + TypeScript.  

---

## 1️⃣ Cài đặt

Clone repo và cài dependencies:

```bash
git clone https://github.com/Kentyumi/bee-giat-say.git
cd bee-giat-say
npm install

2️⃣ Chạy local

Development mode (hot reload)

npm run dev


Xem build đã build local

npm run preview


Truy cập trang: http://localhost:5173


3️⃣ Build & Deploy GitHub Pages

Build project:

npm run build


Deploy lên GitHub Pages:

npm run deploy


Lưu ý: vite.config.ts phải có base: "/bee-giat-say/" (tên repo).

Sau vài phút, web sẽ chạy tại:

https://Kentyumi.github.io/bee-giat-say/
